IADC Code Format Guide

PDC/Fixed Cutter Bits (4 chars)

Format: M433 = Body + Formation + Cutter + Profile

  • M = Matrix, S = Steel, D = Diamond
  • Formation: 1=Soft to 4=Hard (6-8 for Diamond)
  • Cutter: 1=Large(>25mm) to 4=Small(8-10mm)
  • Profile: 1=Fishtail, 2=Short, 3=Medium, 4=Long

Roller Cone/Tricone Bits (3-4 chars)

Format: 517X = Series + Type + Bearing + Feature

  • Series 1-3: Milled Tooth (1=Soft, 2=Medium, 3=Hard)
  • Series 4-8: TCI (4=Soft to 8=Extremely Hard)
  • Feature: X=Enhanced Gauge, Y=Conical Inserts, etc.
