What are Field Mappings?

Field Mappings connect your Excel columns to ERP form fields. When you run a workflow, the automation uses these mappings to know which Excel data goes into which ERP input field. For example: Excel column "LVS MAT NO." can be mapped to the ERP "Material Number" field.
