## Getting Started

### Logging In

1. Navigate to the application URL
2. Enter your username and password
3. Click "Login"

If you forgot your password, contact your administrator.

### Dashboard Overview

After login, you'll see the main dashboard with:

- **Quick Stats** - Key metrics and KPIs
- **Recent Activity** - Latest updates across modules
- **My Tasks** - Items assigned to you
- **Notifications** - Alerts and messages

## Core Modules

### Work Orders

Work orders track all manufacturing and repair activities.

**Creating a Work Order:**
1. Navigate to Work Orders > Create New
2. Select the work order type:
   - **NEW** - New bit manufacturing
   - **REPAIR** - Bit repair
   - **REWORK** - Rework existing bit
   - **RETROFIT** - Upgrade with new design
3. Select the drill bit (if applicable)
4. Set priority and due date
5. Add description and notes
6. Save as Draft or Submit for approval

**Work Order Statuses:**
| Status | Description |
|--------|-------------|
| DRAFT | Being prepared |
| PLANNED | Scheduled for work |
| RELEASED | Ready to start |
| IN_PROGRESS | Currently being worked on |
| QC_PENDING | Awaiting quality inspection |
| COMPLETED | Finished |
| CANCELLED | Cancelled |

### Drill Bits

Track all drill bits throughout their lifecycle.

**Viewing Drill Bit Details:**
1. Navigate to Work Orders > Drill Bits
2. Click on a serial number to view details
3. View history, evaluations, and current status

**Drill Bit Statuses:**
- **AVAILABLE** - Ready for use
- **IN_USE** - Currently deployed
- **IN_REPAIR** - Being repaired
- **SCRAPPED** - End of life

## Field Service

### Customers

Manage customer information and contacts.

**Adding a Customer:**
1. Navigate to Sales > Customers
2. Click "Add Customer"
3. Fill in required fields:
   - Customer code
   - Company name
   - Customer type
   - Contact information
4. Save

### Service Requests

Create and track field service requests.

**Creating a Service Request:**
1. Navigate to Field Service > Requests
2. Click "New Request"
3. Select customer and service site
4. Choose request type:
   - Drill Bit Inspection
   - On-Site Repair
   - Equipment Delivery
   - Technical Support
   - Training
5. Set priority and requested date
6. Submit for approval

## Supply Chain

### Vendors

Manage supplier relationships.

**Vendor Statuses:**
- **PENDING** - Awaiting qualification
- **ACTIVE** - Approved vendor
- **SUSPENDED** - Temporarily blocked
- **INACTIVE** - No longer used

### Purchase Orders

Create and track purchase orders.

**Creating a PO:**
1. Navigate to Supply Chain > Purchase Orders
2. Click "Create PO"
3. Select vendor
4. Add line items with quantities and prices
5. Submit for approval

## Quality & Compliance

### Quality Control

Record QC inspections.

**Performing an Inspection:**
1. Navigate to Quality > Inspections
2. Click "New Inspection"
3. Select inspection type:
   - Receiving
   - In-Process
   - Final
4. Complete checklist items
5. Record result (Pass/Fail/Conditional)
6. Save inspection

### Non-Conformance Reports (NCRs)

Document quality issues.

**Creating an NCR:**
1. Navigate to Quality > NCRs
2. Click "New NCR"
3. Describe the non-conformance
4. Select severity level
5. Submit for review

## My Account

### Updating Profile

1. Click your name in the top right
2. Select "My Profile"
3. Update personal information
4. Save changes

### Changing Password

1. Go to My Profile
2. Click "Change Password"
3. Enter current password
4. Enter new password twice
5. Save

### Notification Preferences

1. Go to My Profile > Preferences
2. Configure email and push notifications
3. Save preferences
